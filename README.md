# CopyTool - macOS 剪贴板历史记录管理器

一个简洁、高效的 macOS 剪贴板历史记录管理工具。

## 功能特点

- 📋 自动记录复制内容（无数量限制，按存储时间自动清理）
- 📝 支持文本内容
- 🖼️ 支持图片内容，支持图片文件自动识别为图片预览
- 📁 支持文件内容，显示文件名
- 👁️ 预览窗口功能，鼠标悬停可预览文本和图片内容
- 🔍 搜索功能，快速查找历史记录
- ✅ 选中历史记录有清晰的视觉反馈
- 🎨 三种类型的内容有不同的图标展示
- 🗑️ 删除单条记录或清空所有历史
- 💾 持久化存储，重启后记录保留
- ⌨️ 可自定义快捷键快速打开
- 🎯 全新设计的设置界面，现代化卡片式布局
- 🔧 丰富的设置选项：
  - 存储时间设置
  - 自定义快捷键
  - 开机启动选项
- 🍎 菜单栏图标，随时访问

## 使用方法

### 1. 启动应用
- 在 Xcode 中运行，或打包后放入应用程序文件夹
- 应用会在菜单栏显示一个文档图标

### 2. 查看历史记录
- 点击菜单栏的文档图标
- 选择"显示历史记录"
- 或使用快捷键（默认 Cmd + Opt + V）

### 3. 使用历史记录
- **点击记录项**：立即复制到剪贴板并关闭窗口
- **点击 × 按钮**：删除单条记录
- **点击"清空所有"**：清除所有历史记录
- **使用搜索框**：输入关键词快速过滤历史记录
- **选中反馈**：点击历史记录会有蓝色的选中效果
- **预览功能**：鼠标悬停在历史记录上可预览文本和图片内容（预览窗口显示在主窗口左侧）

### 4. 设置功能
点击右上角的设置按钮或菜单栏的"设置"选项，可配置：
- **存储时间**：选择历史记录保留时间
- **快捷键**：自定义快捷键，支持预设和自定义录制
- **开机启动**：设置应用是否随系统启动

### 5. 快捷键说明
- 默认快捷键：Cmd + Opt + V（需要辅助功能权限）
- 可在设置中自定义快捷键
- 也可以通过菜单栏图标随时访问

## 系统要求

- macOS 15.7 或更高版本
- Xcode 16.0 或更高版本（开发时）

## 授权辅助功能权限

为了让全局快捷键（失去焦点时也能工作）正常使用，需要授予辅助功能权限：

1. 打开"系统偏好设置"
2. 选择"安全性与隐私" → "隐私"
3. 在左侧选择"辅助功能"
4. 点击锁图标解锁
5. 添加 copytool 应用并勾选
6. 重启应用

## 技术栈

- **语言**：Swift
- **框架**：SwiftUI, AppKit
- **存储**：UserDefaults
- **平台**：macOS
- **开机启动**：ServiceManagement/SMAppService

## 核心功能实现

### 剪贴板监听
- 每 0.5 秒检查一次剪贴板变化
- 使用 NSPasteboard.changeCount 检测变化
- 自动记录文本、图片和文件内容

### 历史记录管理
- 保存记录无数量限制
- 根据设置的存储时间自动清理过期记录
- 使用 UserDefaults 持久化存储
- 支持文本、图片、文件三种类型

### 搜索功能
- 实时搜索过滤历史记录
- 忽略大小写搜索
- 搜索结果空状态提示
- 搜索框带清空按钮

### 选中视觉反馈
- 选中记录有蓝色半透明背景
- 选中时边框变为蓝色并加粗
- 支持选中状态管理

### 文件内容支持
- 检测剪贴板中的文件路径
- 显示文件名而不是通用文本
- 智能文件图标，根据扩展名显示不同图标
- 支持复制文件回剪贴板
- 图片文件自动识别为图片内容并预览

### 预览窗口功能
- 鼠标悬停在历史记录上时显示预览窗口
- 预览窗口显示在主窗口左侧，不遮挡内容
- 支持文本和图片类型的预览
- 快速移动鼠标时预览窗口能正确关闭
- 点击历史记录时立即关闭预览窗口

### 设置功能
- **存储时间**：1天、7天、1个月、永久
- **快捷键**：预设快捷键 + 自定义录制
- **开机启动**：使用现代 SMAppService API
- **向后兼容**：旧版本 macOS 也支持
- **全新设计**：卡片式布局，更美观易用

### 用户界面
- 菜单栏状态项（NSStatusItem）
- 弹出式窗口（NSPopover）
- SwiftUI 列表显示历史记录
- 设置窗口（NSWindow）

## 打包发布

### 1. 使用 Xcode Archive
1. Product → Archive
2. 在 Organizer 中选择归档
3. 点击"Distribute App"
4. 选择"Copy App"选项

### 2. 手动构建
1. 在 Xcode 中设置 Scheme 为 Release
2. Product → Build
3. 在 Build/Products/Release 文件夹中找到 .app 文件
4. 复制到应用程序文件夹

### 3. 使用脚本构建
1. 运行 `./build_app.sh` - 构建 Release 版本的应用
2. 运行 `./create_dmg.sh` - 创建 DMG 安装包
